## XML 转义字符

| 小于     | <    | &lt;       |
| -------- | ---- | ---------- |
| 小于等于 | <=   | `&lt;=`    |
| 大于     | >    | `&gt;`     |
| 大于等于 | >=   | `&gt;=`    |
| 不等于   | <>   | `&lt;&gt;` |
| 与       | &    | `&amp;`    |
| 单引号   | '    | `&apos;`   |
| 双引号   | "    | `&quot;`   |

## CDATA 转义区段

```xml
大于等于  <![CDATA[ >= ]]>
小于等于  <![CDATA[ <= ]]>
不等于    <![CDATA[ <> ]]>
```
